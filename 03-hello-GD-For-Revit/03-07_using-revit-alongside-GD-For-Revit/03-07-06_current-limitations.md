# Current Limitations

The _`Data.Remember`_ node has the following known limitations:

* It currently only works with the following data types:
  * Strings
  * Numbers
  * Booleans
  * All Dynamo Geometry Types
  * Date-Time
  * Location
  * Images
  * Lists
  * Dictionaries
  * Nested Lists
  * Nested Dictionaries
  * Mixed Lists and Dictionaries
